# ATM Simulation System

## Overview

ATM Simulation System is a console-based Python application designed to simulate basic banking operations. The system allows users to deposit money, withdraw funds, check account balance, and view transaction history through a simple menu-driven interface. This project helps in understanding conditional statements, functions, exception handling, and transaction management in Python.

## Features

* Deposit Money
* Withdraw Money
* Balance Inquiry
* Transaction History
* Exception Handling
* Menu-Driven Interface

## Technologies Used

* Python 3.x
* Visual Studio Code
* Git & GitHub

## Concepts Used

* Variables and Data Types
* Conditional Statements
* Loops
* Functions
* Lists
* Exception Handling
* Input Validation
* Transaction Management
* Menu-Driven Programming
